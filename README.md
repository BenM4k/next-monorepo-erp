## 📖 About Quantiva

**Quantiva** is a modern enterprise solution designed to unify financial management, operations, and analytics into a seamless digital experience.

It replaces fragmented tools with a **centralized system** that improves efficiency, accuracy, and decision-making.

### 🎯 Vision

To build a **powerful, accessible ERP platform** tailored for businesses in emerging and global markets.

---

## 🔧 Installation

```bash
git clone https://github.com/your-username/quantiva-erp.git
cd quantiva-erp
npm install
```

Create a `.env` file and configure:

```env
DATABASE_URL=your_postgresql_connection
JWT_SECRET=your_secret_key
```

Run the app:

```bash
npm run dev
```
